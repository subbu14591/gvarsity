	<h4>Bug Fixes</h4>
	<ul>
		<li>7424: IsSuperUser in dbo.Users is reset to false when user logs in with AD Account.</li>
		<li>7610: Redirect to Original Page after Authentication.</li>
		<li>7620: Authentication ticket has hardcoded timeout.</li>
		<li>7640: Reinstall of provider causes web.config error.</li>
		<li>7703: Deleted users not re-enabled when logging in manually.</li>
		<li>7739: Profile cleared out on login.</li>
	</ul>
	<p class="Owner">
		To run the provider under IIS 7.x in integrated mode two commands are needed to be run from a command prompt on the server.
		<ul>
			<li>%windir%\system32\inetsrv\appcmd unlock config /section:anonymousAuthentication</li>
			<li>%windir%\system32\inetsrv\appcmd unlock config /section:windowsAuthentication</li>
		</ul>
